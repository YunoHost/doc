*OpenTracker* is a Bittorrent tracker.

### Disclaimers / important information

### Configuration

`udp://serverIP:6969`

By default the tracker will run on 6969 port on both UDP and TCP.

Visit `http://serverIP:6969/stats` to view the tracker stats.
