## Comment demander de l'aide

<Columns> 
  <Column className='text--left'>
  
    :::tip
    - **Soyez courtois**: l'aide apportée dans les salons de discussion ou sur le forum est fournie par nos utilisateurs et contributeurs, tous bénévoles.
    - **Expliquez le contexte** de votre demande de manière complète mais concise.
    - Il est primordial de **partager les journaux d'erreurs avec YunoPaste** (boutons verts dans votre webadmin, puis partagez le lien vers la page générée).
    - **Soyez patient**, car vos messages peuvent ne pas être vus instantanément.
    :::

  </Column>
  <Column className='text--left'>

    :::danger
    - Ne posez pas de questions sur la solution que vous avez essayée, mais plutôt sur votre problème réel (voir [problème XY](https://xyproblem.info/)).
    - Évitez les phrases telles que « le serveur ne veut pas », « l'application ne se charge pas » ou d'autres phrases dénuées de sens.
    -  Ne raccourcissez pas les messages d'erreur, mais ne collez pas non plus un mur de code illisible et non formaté. (Utilisez [YunoPaste](https://paste.yunohost.org))
    - Ne posez pas votre question sous forme de sondage (« quelqu'un d'autre a-t-il réussi à... ? »).
    :::

  </Column>
</Columns>

Ce salon de discussion est disponibles via
