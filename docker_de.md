# Docker und YunoHost

<div class="alert alert-danger">
<b>
YunoHost unterstuetzt seit Version 2.4 keine offiziellen Docker images,
da YunoHost systemd benoetigt, welches von Docker nicht nativ unterstuetzt wird.
</b>
</div>

## Community images

Hier sind von der Community erstellte Images, verfuegbar auf dem Docker Hub:

  * AMD64 (classic)
    * https://hub.docker.com/r/domainelibre/yunohost/ (YunoHost v4+)
  * I386 (Alte Computer)
    * https://hub.docker.com/r/domainelibre/yunohost-i386/ (YunoHost v4+)
  * ARMV7 (Raspberry Pi 2/3 ...)
    * https://hub.docker.com/r/domainelibre/yunohost-arm/ (YunoHost v4+)
  * ARMV6 (Raspberry Pi 1)
    * https://hub.docker.com/r/tuxalex/yunohost-armv6/ (old yunoHost version)
